# VKP Technologies Platform

Production-ready IT business platform for VKP Technologies.

## Features

- Website for IT services and hardware sales
- Product catalogue
- Razorpay payment integration
- Invoice PDF generation
- Stock management ERP
- Dealer login system
- Admin panel
- CRM lead capture
- WhatsApp chatbot automation
- n8n workflow
- Firebase backend support

## Tech Stack

- React
- Vite
- Node.js
- Firebase
- Razorpay
- WhatsApp Cloud API
- n8n

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Backend

```bash
npm run api
```

## Environment Variables

Copy `.env.example` to `.env` and update Razorpay, Firebase, and WhatsApp credentials.
